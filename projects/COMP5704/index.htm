Project Outline: With the recent development of Convolutional Neural Network, it has been used to solve several Computer Vision problem. Most of these CNN as generative model, produces the same output as input which is known as object reconstruction. But the aim of this work is to leverage the power of these generative models to produce new object rather than the same object. For this, several matrix addition operations will be executed between the results of convolutional neural network's layer. To make the program computationally faster, we will take the advantage of CUDA platform to execute these operations inside GPU using multiple threads.
